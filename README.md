# Tabela Projeto :sparkles:

A simple web application for managing a product table or shopping list. This project allows users to view and interact with a table of products, with modern styling and responsive design.

## Features

- Modern header with title "Lista de Compras"
- Responsive table design with hover effects
- Column highlight (radar effect) on hover
- Clean and modern UI with shadows and spacing
- Google Fonts integration (Roboto)

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/michaelkalleby-ui/Tabela-Projeto.git
   ```

2. Open `index.html` in your web browser.

## Files

- `index.html`: Main page with the product table
- `produtos.html`: Additional products page
- `style.css`: Main stylesheet
- `produtos.css`: Styles for products page
- `script.js`: JavaScript for interactivity
- `TODO.md`: Project tasks and progress
